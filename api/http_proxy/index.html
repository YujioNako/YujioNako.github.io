HTTP Proxy

使用方法：

1. 下载该配置后导入到软件中（不推荐）
2. 直接将配置地址：https://www.pro-ivan.com/api/http_proxy/http_proxy.yaml 填入软件（推荐）

注意事项：

不建议设置自动更新配置，因为云端内容更新较快且变化较大，可能影响您在使用中的体验。建议您只在需要的时候（如所有通道连接质量均不佳、通道缺失等）更新配置文件。
配置仅供支持Clash YAML URL的软件（如Clash for Windows，Shadowrocket等）使用。本配置提供的通道支持https访问，但大部分通道不支持流媒体信息传输。
